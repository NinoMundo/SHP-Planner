# SHP Planner

SHP Planner is a Node.js web app that helps digital nomads and expats plan their Paraguay residency process through Sweet Home Paraguay.

## Features
- Collects key residency inputs
- Live planning snapshot UI
- Ready for roadmap generation logic

## Local setup

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Environment

Set these in `.env` (do not commit):

```
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
PDF_TEMPLATE_PATH=
SES_REGION=
SES_ACCESS_KEY_ID=
SES_SECRET_ACCESS_KEY=
SES_FROM_EMAIL=
```
